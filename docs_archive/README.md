# Thư mục Lưu trữ Tài liệu Sinh tự động (Archive)

Thư mục này chứa các tài liệu, sơ đồ và hướng dẫn kỹ thuật được Bot AI (Antigravity) tự động sinh ra trong quá trình Thiết kế Hệ thống và Lập kế hoạch Sprint 2 cho dự án NP Education.

Vì các tài liệu này có thể làm "loãng" hoặc rối mắt (clutter) kho lưu trữ mã nguồn chính (GitHub repository), chúng được di chuyển hết vào thư mục `docs_archive/` để lưu trữ riêng.

## Danh sách tài liệu:
1. Sơ đồ Kiến trúc & Database (`architecture.png`, `database.png`)
2. Các bản nháp thiết kế Core (`architecture_design.docx`, `database_design.docx`, `rbac_design.docx`)
3. Hướng dẫn thiết lập Server (`backend_setup.docx`, `api_spec.docx`)
4. Kế hoạch & Hướng dẫn Demo (`ban_thuyet_trinh.docx`, `huong_dan_demo.docx`, `team_tasks.docx`)

*Thư mục `/documents` gốc của dự án hiện tại chỉ còn chứa các file Excel/Yêu cầu gốc ban đầu.*
